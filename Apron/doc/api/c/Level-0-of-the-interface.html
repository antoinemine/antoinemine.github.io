Level 0 of the interface ¶

This interface of level 0 is defined in ap_global0.h.

Unless there exists specific reasons for not doing so, we advise the user to use the level 1 of the interface (see Level 1 of the interface). The level 0 is intented for implementors who wants to connect a new library/abstract domain, or who want to build a composite domain from existing ones.

  • Dimensions and related operations (ap_dimension.h)
  • Linear expressions of level 0 (ap_linexpr0.h)
  • Linear constraints of level 0 (ap_lincons0.h)
  • Generators of level 0 (ap_generator0.h)
  • Tree expressions of level 0 (ap_texpr0.h)
  • Tree constraints of level 0 (ap_tcons0.h)
  • Abstract values and operations of level 0 (ap_abstract0.h)
